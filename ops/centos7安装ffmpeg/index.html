Centos7安装ffmpeg

安装 EPEL

yum install epel-release -y

安装Nux Dextop 库

rpm --import http://li.nux.ro/download/nux/RPM-GPG-KEY-nux.ro
rpm -vhU http://li.nux.ro/download/nux/dextop/el7/x86_64/nux-dextop-release-0-1.el7.nux.noarch.rpm

安装 FFmpeg

yum install ffmpeg ffmpeg-devel -y

安装验证

ffmpeg -version
  • 安装成功
