  • 法国的一个护工因车祸成植物人
  • 在医院里吊管子维系生命
  • 卧床11年间依赖人工喂饲维系生命
  • 他的妻子认为不应再继续这种无用的治疗
  • 医生建议将喂食管拔除
  • 这个人将会在一周以内死去
  • 便是该男子的父母坚决反对
  • 然后开始打官司
  • 不同级别有不同级别的判决
  • 司法天平不断的在其妻子和父母方来回倾斜
  • 这个案子的在双方看都是有理的
  • 妻子放弃了11年的无用治疗
  • 父母不愿意放弃那万分之一的可能
  • 对法律不是很懂
  • 但是与我个人来说
  • 如果有天我出意外了
  • 无法自己表达真实想法时
  • 当我的生死置与其他人的决定时
  • 我希望那个人是值得信任的
  • 我希望那个人是这个世上最信任的人
  • 这对我很重要
  • 不知道别人是怎么想的
  • 我有很长一段时间
  • 在确定和寻找这样一个人
  • 这也是为什么会经常感到孤独的原因

最值得信任的人

  • 结论：父母>子女>兄弟姐妹

亲情:父母

  • 无私奉献的父母
  • 一起长大的兄弟姐妹
  • 日渐长大的小孩
  • 大多数的人在世上最信任的人应该都是父母
  • 把我们抚养长大
  • 很多时候你都能明显感受到
  • 如果有可能他们会放弃自己去救你
  • 这种亲情羁绊无比清晰
  • 是绝对值得信赖的
  • 但是：
  • 1.父母正在老去，迟早会离我们而去。
  • 2.有兄弟姐妹的，父母有偏心的
  • 第一种是客观现实无法改变
  • 第二种非常可悲
  • 父母对一个孩子明显好过另一个孩子
  • 会很容易伤到另一个孩子
  • 很难理解这种缺心眼的行为
  • 同样是自己的娃
  • 为啥要分等级
  • 但是人嘛总有自己的好恶
  • 以及传统文化观念
  • 比如重男轻女
  • 其实在我小时候
  • 父母也会表现出来对我比对我姐好一些
  • 说不上来具体的事
  • 所以我那个强悍的姐总是为这事哭闹
  • 等我们上了学稍微懂了事
  • 我也会刻意让着我姐
  • 因为说是我姐
  • 其实只比我大1岁
  • 很多时候我都把自己放在哥的地位上
  • 而且我越长大越皮
  • 父母看着也嫌了
  • 所以父母只要不过分偏心
  • 亲情的信任感会很强烈

亲情：子女

  • 为人父母后
  • 会信任自己的娃吗
  • 大多数父母与子女的关系
  • 都是单方面的父母对子女的付出
  • 说不信任是不可能的
  • 可是面对越来越长大的孩子
  • 越来越叛逆
  • 开始搞些超出你理解范围的事后
  • 你还会信任你的孩子吗
  • 会也不会
  • 会信任他的心
  • 但是不会信任他的行为
  • 有一种特殊情况
  • 假如父母离异了
  • 虽然做为共同抚养人
  • 但总归是由一方抚养长大
  • 这时候长时间不在身边的孩子
  • 他感受到的是不完整的来自父母的爱
  • 他至少不是特别信任那个远离他的父母
  • 而作为离开的那个父母
  • 怎么处理这段关系呢
  • 别放下这段关系
  • 不是为了谁
  • 亲情关系也没想象中的牢固
  • 一旦放下了也会淡的
  • 明白清晰的表达表现出来
  • 把对孩子的爱和信任
  • 感示出来
  • 那是你带到世界上来的生命
  • 让他信任你
  • 让他依赖你的情感
  • 这是本来就应该做的事
  • 并不是因为你离开了那段婚姻
  • 就可以放弃的

亲情:兄弟姐妹

  • 只有一个姐姐
  • 我对她的信任感不高
  • 从小就这样
  • 这就非常奇怪
  • 一起从小长大
  • 也没什么矛盾
  • 我们在大多数事上都互相帮衬
  • 但是人的感觉就是这么奇怪
  • 我很少和她说我的事
  • 她其实离我的生活很远
  • 她不知道我认识什么样的人
  • 有时候我在想
  • 如果换成一个哥哥或弟弟
  • 我们关系一定非常好
  • 而我和姐性格差异太大
  • 很多时候没有共同语言
  • 虽然不是很信任
  • 只是相对于父母的亲情来说
  • 比起其他人
  • 我还是很信任我姐的
  • 这是不用置疑的

爱情

  • 最理想的关系
  • 如果你对伴侣的信任
  • 超过了对父母的信任
  • 那么至少对你来说
  • 这是一段非常美好和重要的爱情
  • 如果有一个人能这样信息你
  • 别背叛他
  • 别伤害他
  • 信任一旦崩塌
  • 绝对会一地鸡毛
  • 可能越是信任的人越经不起背叛
  • 也有可能
  • 他就失去了信任别人的能力

爱很容易信任难

  • 当知道自己没那么优秀时
  • 会发现找到一个喜欢的异性并不是很难
  • 喜欢一个人
  • 或者爱上一个人
  • 并没有那么难
  • 人之一生
  • 可能会喜欢上很多个人
  • 但是信任呢
  • 会信任这个人吗
  • 你自己在这段关系里是否值得信任呢
  • 美貌财富地位可以让人快速的看上一个人
  • 智慧和有趣会让人真正的喜欢上一个人
  • 但信任是感情的可以源源不断的源头
  • 信任是所有纷争矛盾的解药
  • 穿越人性的阴暗
  • 有那么一个人
  • 你可以放心的把自己不好的说给她听
  • 把自己的心思袒露在她面前不会觉得难堪
  • 有那么一个人
  • 她愿意分享她的过往，好的和坏的
  • 她愿意表达她的真实想法
  • 如果有这样的人
  • 好好珍惜吧
  • 如果有天你不稀罕了
  • 请妥当善良的离开
  • 不要伤害他

怎么做

  • 亲情的信任是与生俱来的
  • 而爱情的信任
  • 是需要双方默契和坦诚
  • 是个培养的过程
  • 讲真话是最基本的要求
  • 对我而言
  • 在陌生关系前
  • 不想也不愿意暴露自己的真实需求
  • 我是怎么想的
  • 我为什么这么做
  • 我有哪些缺点
  • 不愿意说或者乱说
  • 但是一旦确定关系了
  • 还是很希望对方了解我的真实情况是什么样子的
  • 希望对我能有多一些了解
  • 不介意暴露我的弱点
  • 更多的希望对方能了解我的生活习性
  • 因为默契
  • 在相处中会让我很高兴
  • 而分歧
  • 总是让人沮丧的
  • 为什么要这样呢
  • 坦诚一点不好吗
  • 我也曾试图理解那些在伴侣关系中谎话连篇的人
  • 也没有做什么错事
  • 只是在掩饰一些不愉快的事
  • 或者是不想产生过多的解释
  • 甚至是不想产生分歧
  • 千奇百怪的想法吧
  • 但是…
  • 这段关系你想长期维护吗？
  • 你的智慧足够支撑每次说谎都能天衣无缝吗？
  • 你准备信任对方吗？
  • 不过是一次次在消费对方的信任度而已
  • 自欺欺人还不自知

友情

  • 你可能会遇到
  • 投缘的朋友
  • 靠谱的同事
  • 热情的邻居
  • 值得信任吗
  • 值得的
  • 信任的度呢？
  • 当涉及利益冲突的时候
  • 你还能信任的你的朋友们吗
  • 我愿意相信
  • 但不会无条件的相信的
  • 这种信任它是有底线的
  • 至少不会主动暴露自己的一些真实想法
  • 更多的信任
  • 是相信朋友或同事的
  • 能力，品性
  • 保持最大善意
  • 去对待别人
  • 做个值得信任的人
  • 信任那些认可的朋友同事和各种各样的人
  • 同时留有余地
  • 保持在合适合理的范围

结论

  • 信任度：父母>=爱人>子女>伴侣>兄弟姐妹>挚友>同事>亲戚
  • 如果我意外成植物人了，决定我生死的一定要是我爱的人，也是爱我的人。否则这一生就太失败了。
