mssql 内外网不同端口访问

作者：[renyb ]

1、配置工具中  tcp/ip添加1433,8000

2、use [master]
GO
CREATE ENDPOINT TSQL_Settlement
STATE = STARTED
AS TCP ( LISTENER_PORT = 8000 )
FOR TSQL ()

3、USE [master]

GRANT Co nNECT o n ENDPOINT::TSQL_Settlement TO [public]
GO

4、做个校验
SELECT *
FROM sys.endpoints;

  PS: 内网可以用1433端口。外网防火墙过滤1433，通过开放的mssql端,8000来访问。
