MySQL集群7.2版发布 实现每分钟10亿次查询

作者：[网上资料 ]

据国外媒体报道，甲骨文公司日前已经宣布正式发布MySQL Cluster 7.2版，作为商用的MySQL Cluster电信运营商级版本的一部分，它包括了管理工具，产品认证和24x7全球支持。
新的基准测试程序显示了MySQL Cluster 7.2版在保持99.999％可用性的同时，支持要求最苛刻的网络和电信工作负荷的能力，同时甲骨文公司还声称MySQL Cluster 7.2版可以实现每分钟10亿次查询（每秒1760万查询），跨越8个商用英特尔x86服务器向外扩展，采用了全新的NoSQL NDB C ++ API。
为了通过自动分片（auto-sharded）和多主架构（multi-master architecture）显示原生写法的可扩展性，MySQL Cluster 7.2版实现了每分钟1.1亿次更新（每秒钟180万次），开发人员再无需复杂地将其在应用程序级别的数据库进行手动分片（分区）。
MySQL Cluster 7.2版在不影响稳定性和高冗余的前提下，实现了这一级别的网络吞吐量（Throughput），所有更新在数据节点之间被同步复制，确保集群保持99.999％的可用性，在遇到中断故障或计划进行维护活动的情况下，没有数据会丢失。
此外，MySQL Cluster 7.2版还提供了丰富的SQL功能，并保证了所有数据的完整性，满足了用户对一个兼容ACID的关系数据库的期望，使用户能够对他们的OLTP数据集执行复杂的实时查询。
