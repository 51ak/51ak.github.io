SQL Manager 2010 for Mysql 破解

作者：[网上资料 ]
最近用了Windows Server 2008系统，结果导致了很多程序不得不用最新版的来避免兼容性问题。用原来的Sql manager 2007说什么也连接不上，没有办法，下了Sql Manager 2010 成功连接，可是问题又来了，没有破解，上国外网站找了半天也没有个免费下载的，国内的更是连个付费的都没有，googel 一下找到破解方式
1  先安装Sql manager 2010 .建议到官网下载
   http://www.sqlmanager.net/PRoducts/studio/MySQL/quick-download
2  下载Sql manager 2005 版的破解
   http://download.csdn.net/source/2211509
3 安装完2010后,然后用安装后的SQL Manager目录下(如:C:\Program Files (x86)\EMS\SQL Studio for MySQL\SQL Manager )用2下载的MyManager.exe 代替。即可实现破解版（仅限于MyManager.exe破解，其他仍为30天的试用期）。
