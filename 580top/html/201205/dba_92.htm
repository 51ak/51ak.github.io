如何从命令提示符安装 sql server 2008

作者：[renyb ]

参考文献：  联机帮助  如何从命令提示符安装 SQL Server 2008

A、将SQL Server 2008的安装文件复制到D:\tools\SQL2008

B、 提取补丁包内的文件：
命令行下运行：SQLServer2008SP3-KB2546951-x64-CHS.exe /x:d:\SQLServer2008，将补丁中的文件释放到d:\SQLServer2008\PCU目录下


1.复制 Setup.exe 和 Setup.rll 提取的协调股的位置原始来源媒体位置

robocopy D:\tools\SQL2008\PCU D:\tools\SQL2008 Setup.exe

robocopy D:\tools\SQL2008\PCU D:\tools\SQL2008 Setup.rll

2.复制所有文件而不是文件夹， 除了Microsoft.SQL.Chainer.PackageData.dll ，从D:\tools\SQL2008\PCU\ 到D:\tools\SQL2008 \ 来更新原始档案。

robocopy C:\SQLServer2008_FullSP1\pcu\x86 C:\SQLServer2008_FullSP1\x86 /XF Microsoft.SQL.Chainer.PackageData.dll

robocopy D:\tools\SQL2008\PCU\x64 D:\tools\SQL2008\x64 /XF Microsoft.SQL.Chainer.PackageData.dll

robocopy C:\SQLServer2008_FullSP1\pcu\ia64 C:\SQLServer2008_FullSP1\ia64 /XF Microsoft.SQL.Chainer.PackageData.dll


3. 确定您是否拥有defaultsetup.ini在下列地点：
C:\SQLServer2008_FullSP1\x86
C:\SQLServer2008_FullSP1\x64
C:\SQLServer2008_FullSP1\ia64
如果你有一个defaultsetup.ini ，增加下列内容到PCUSOURCE=”.\PCU” ，如下：
;SQLSERVER2008 Co nfiguratio n File
[SQLSERVER2008]
PCUSOURCE=".\PCU"

4、 DOS窗口执行

D:\tools\SQL2008\setup.exe  /q /ACTIo n=Install /SQLSVCPASSWORD="" /AGTSVCPASSWORD="" /ISSVCPASSWORD="" /SAPWD="*******"  /Co nfiguratio nFile=D:\tools\Co nfiguratio nFile.ini
