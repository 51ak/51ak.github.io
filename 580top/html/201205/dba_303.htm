Oracle10安装apex3.2步骤

作者：[网上资料 ]
1.安装oracle10，我安装的是Release 10.1.0.2.0，可以在安装时建库，也可以安装后通过dbca创建一个简单数据库
　　2.安装companion光盘中的第三项，oracle database 10g companion products 10.2.0.1.0，和数据库安装在不同的目录下
　　3.下载apex 3.2，解压缩
　　4.先安装数据库，后安装httpserver的话，环境变量可能会有点问题，可以先置一下环境变量
set ORACLE_HOME=D:\oracleset J2EE_HOME=%ORACLE_HOME%\j2ee\homeset ORACLE_SID=billset PATH=%ORACLE_HOME%\jdk\bin;%ORACLE_HOME%\bin;%PATH%set LD_LIBRARY_PATH=%ORACLE_HOME%\lib;%ORACLE_HOME%\ctx\lib 　　路径要根据安装实际情况修改
　　4.进入apex目录，以sys登录sqlplus，执行 @apexins SYSAUX SYSAUX TEMP /i/
　　SYSAUX SYSAUX TEMP 是指定的三项表空间，/i/是指定的图片的虚拟路径
　　以前版本的命令参数是 @apexins PASSWD SYSAUX SYSAUX TEMP /i/，3.2版做了修改，passwd用@apxchpwd.sql单独修改。
　　5.安装完成后把apex下的images目录拷贝到httpserv下的\Apache\Apache目录下
　　6.配制httpserv下的Apache\modplsql\conf\dads.conf，典型的配制如下#
# ============================================================================# mod_plsql DAD Configuration File # ============================================================================# 1. Please refer to dads.README for a description of this file # ============================================================================Alias /i/ "D:/oraclehttp/Apache/Apache/images/" ##图片虚拟路径和实际路径AddType text/xml xblAddType text/x-component htc<Location /pls/apex>Order deny,allowPlsqlDocumentPath docsAllowOverride NonePlsqlDocumentProcedure wwv_flow_file_manager.process_downloadPlsqlDatabaseConnectString localhost:1521:bill ServiceNameFormat ##数据库sidPlsqlNLSLanguage "SIMPLIFIED CHINESE_CHINA.AL32UTF8"PlsqlAuthenticationMode BasicSetHandler pls_handlerPlsqlDocumentTablename wwv_flow_file_objects$PlsqlDatabaseUsername APEX_PUBLIC_USERPlsqlDefaultPage apexPlsqlDatabasePassword oracle ##数据库APEX_PUBLIC_USER用户密码Allow from all</Location> 　　7.在可以访问之前还有些要做的
　　（1）lstrctl启动监听
　　（2）数据库APEX_PUBLIC_USER用户解锁
　　（3）修改APEX_PUBLIC_USER用户密码和配制文件一致8.启动httpserv，可以访问了http://localhost：7777/pls/apex
　　8.启动httpserv，可以访问了
