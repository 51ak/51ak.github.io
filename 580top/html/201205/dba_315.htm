Oracle安装方法及简介

作者：[网上资料 ]
1. Oracle数据库：客户端/数据库服务器 使用Oracle10g2. 安装：   配置: Database Configuration Assistant 配置数据库实例         Net Configuration Assistant  配置数据库服务器的监听         isqlplusctl start  启动isqlplus的服务(如果需要使用isql*Plus)   服务: OracleService实例名  数据库实例的服务         OracleOraDB10g_home1TNSListener  数据库服务器的监听服务3. Oracle的常用管理工具：   1) oem:企业管理器 -->DBA  http://IP:1158/em   2) sql*Plus:最常用的工具. Oracle的客户端工具   3) isql*Plus:通过浏览器来操作数据库服务器  http://ip:5560/isqlplus4. Oracle中的几个用户：     sys/oracle -->角色sysdba     scott/tiger-->Normal普通用户5. sql*Plus的常用命令：  1) 打开sql*Plus: 在命令中使用 sqlplus  2) 打开并连接上服务器：sqlplus [用户名/密码][@服务名] [as 角色名]       如：sqlplus scott/tiger           sqlplus sys/oracle as sysdba  3) 连接命令：      conn[ect] 切换连接      disc[onnect] 断开连接      passw[ord]  更改密码      exit,quit 退出sql*Plus   4) 编辑命令：l[ist], /   5) 文件操纵命令：      save 文件全路径名 把缓冲区中的命令保存到指定文件      get 文件全路径名 把指定文件中的命令加载到缓冲区      @文件路径名 执行指定文件中的SQL      ed 文件路径名 编辑缓冲区或指定文件   6) 查看sql*Plus命令的帮助：      help index      help 命令名   7) 基本命令:      show user 查看当前的连接用户;      set pagesize 值(14) 设置每页显示的行数      set linesize 值(80) 设置一行中显示的字符数      desc 表名  查看表的结构6. 常用数据类型  1) CHAR(长度)：定长字符串      2000  2) VARCHAR2(长度)：变长字符串   4000  3) NUMBER[(总位数,小数位)]：数字，整数和小数。默认是38位精度  4) DATE：日期和时间，秒值  5) TIMESTAMP：时间戳。日期和时间，时区。小数位秒  6) CLOB 大字符数据     BLOB 大二进制数据     BFILE 文件指针7. SQL：结构化查询语言。SQL1999。大小写不敏感。建议关键字用大写，其它用小写。  1) 分类：     ★a) 查询语句：SELECT     ☆b) DML：数据操纵语言。定义修改内容的语句。INSERT、DELET、UPDATE、TRUNCATE     ☆c) DDL：数据定义语言。定义数据库对象的语句。CREATE、ALTER、DROP、RENAME       d) DCL：数据控制语言。定义对用户访问权限的语句。GRANT、REVOKE     ☆e) TCL：事务控制语言。定义事务控制语句。COMMIT、ROLLBACK、SAVEPOINT |
