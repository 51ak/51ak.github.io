错误处理：指定的包无法完成运行即已停止

作者：[51ak ]
错误ID:8885
严重程度8（严重报警)
总报警次数345
错误类型:App
错误来源:SQLISPackage100
错误描述: Package
处理建议:指定的 Integration Services 包无法完成运行即已停止。
可能的原因
1.导致包停止运行的可能原因有很多。此消息可能因以下问题之一而出现：
2.在开发计算机上指定的数据源、文件或凭据在已部署包的计算机上不可用或无效。
3.在开发计算机上可用的数据提供程序在已部署包的计算机上不可用。或者，在 32 位版本中可用的数据提供程序在包作为 64 包运行时不可用。
在 SQL Server 代理作业中计划执行包时，包要求的权限不足。
用户操作
1.如果错误为意外错误，请查看 Integration Services 包的故障排除建议。。
2.如果导致包停止运行的错误属于偶尔发生的预期行为，请尝试下列选项之一：
a.考虑使用错误处理程序处理该错误。有关详细信息，请参阅 Integration Services 事件处理程序。
b.考虑使用检查点从特定点重新启动包。有关详细信息，请参阅在包中使用检查点。
c.考虑使用事务确保数据的完整性。有关详细信息，请参阅使用事务确保数据完整性。
