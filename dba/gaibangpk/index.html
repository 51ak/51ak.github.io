丐帮PK赛

[ 丐帮pk赛：帮派里选举出来的最睿智的两个人，进行一对一pk，分为P方和K方，互相出题，答题时长不限，方法不限，只要答出来了，就算得1分，统计最终得分。 ]

【第一轮.地铁站走楼梯问题】

题目:从知春路地铁站走楼梯上来，共有30级台阶，阿怂每次上台阶要么是1层，要么是2层，问：有多少种方法阿怂可以爬完这30级台阶？

  • 出题人:P
  • 答题人:K
  • 答题时间:用时17分钟
  • 答题结果:正确
  • 正确答案:1346269
  • 解答:每一层楼梯的方法不是他决定的,是他上一级台阶有几种方法和上上一级台阶有几种方法,加一块就是他有几种方法.

【第二轮.病毒感染率问题】

题目:假设有一种病毒，感染率为1%，针对病毒检测，误诊率为1%,如果医生通知你是感染者，你真正感染的概率是多少？

  • 出题人:K
  • 答题人:P
  • 答题时间:用时2分钟
  • 答题结果:失败! 答成了99%
  • 正确答案:50%
  • 解答:假设有10000人，感染率为1%,那么其中有100为感染，9900位健康,其中100感染者里，有99位真感染，1人误诊，9900位健康者里，有99位因为误诊是感染的，9801位是健康的。于是：被告知感染的人里，有99位真正的感染者和99位健康的人。你感染了被诊断出来的几率是0.01*（1-0.01）=0.0099，假如你没感染被诊断出来的几率是0.99*0.01=0.0099所以实际上依然健康的概率为0.0099/（0.0099+0.0099）=0.5 贝叶斯理论

 

【第三轮.古惑仔收保护费问题】

题目:古惑仔阿怂要到一条街上收保护费，老大交待这次不能同时收相邻的两个捕子的保护费，收了第1号铺子的就不能收第2号，可以跳过去收3或者4号。假如从1号铺子开始每个铺子能收到的保护费分别是4, 3, 5, 4, 5, 23, 7, 8, 9, 7, 7, 8, 12, 15, 17, 13, 4, 9, 12, 17, 13, 5, 0, 7, 14, 7, 9, 8, 9, 30 元，问：阿怂最多能收到多少元钱？

  • 出题人:P
  • 答题人:K
  • 答题时间:用时18分钟
  • 答题结果:166 算错了一个数
  • 正确答案:167
  • 解答:当收到第N个铺子时，可以选择收和不收，
    • 选择收，那么这个铺子时的最大金额就是N-2间铺子的最大金额+第N间铺子
    • 选择不收，那么收到这个铺子时的最大金额就是第N-1间铺子的最大金额
    • 所以在N号铺子时，怂哥最多能拿到的钱就是上面两个值取个最大值，把这个值人做个记号，然后接着去下个铺子做同样的事，结果就一路推出来了。
    • 在第N号铺子，只需要回头看看N-2，N-1号铺子门口的最大金额标记和这个铺子的钱就知道这个铺子的记号。

【第四轮.古惑仔收保护费问题】

题目:参见第三题，还是这个古惑仔阿怂要到一条街上收保护费，跟刚才的不一样的是刚才的一条街的30个铺子是直的，这次是个圆形，也就是第1号铺子和第30号铺子是挨着的，问阿怂这次最多能收到多少钱

  • 出题人:P
  • 答题人:K
  • 答题时间:用时5分钟（中间有中断，为预估时间）
  • 答题结果:正确 165
  • 正确答案:165
  • 解答:参照第三题的答案和算法，因为首尾相连了，这时候就必须扔掉一个铺子，要么是1 ，要么是30号铺子，因为很明显这次的30铺是大户附近(28-30,1-3)都没有大铺可以抵销扔掉30号铺子的损失，所以这题的解法是把1号铺子扔了，把2号铺子当成1号铺子来处理。

 

【第五轮.盒子里的小球问题】

题目:桌子上有5个盒子，从左到右分别装有8,4,2,1,5个相同的小球。每次操作都从其余4个盒子中各拿出1个小球，放入小球数最少的盒子中；如果有2个盒子中小球数相同，就认为其中左侧盒子的小球数较少；如果某个盒子中只有1个小球，则不从其中拿出小球。经过10000次操作后，第1个盒子中还有多少个小球？

  • 出题人:K
  • 答题人:P
  • 出题时间:10:03
  • 答题时间:10:39(用时36分钟)
  • 答题结果:3正确,最后的数组是【3,4,2,6,5】
  • 正确答案:3
  • 解答:没开始操作前：小球数为8,4,2,1,5.第1次操作后：小球数为7,3,1,5,4，第2次操作后：小球数为6,2,5,4,3,第3次操作后：小球数为5,6,4,3,2,第4次操作后：小球数为4,5,3,2,6,第5次操作后：小球数为3,4,2,6,5,第6次操作后：小球数为2,3,6,5,4,第7次操作后：小球数为6,2,5,4,3。可以看出第2次与第7次操作后，5个盒子中的小球数完全相同，出现了重复的周期规律，由于10000=5*2000，故第10000次操作后，盒子中的小球数与第5次操作后相同
  • 备注：抽屉原理

【第六轮.二狗子坐哪问题】

题目:张师傅的新片《1秒钟》上映，顺义的博纳国际影城8号厅共有7排，每排10个座位。今天这场爆满，票都卖完了。排队进场时，第1个进场的顾客张老汉票根上的座位号特别模糊，认不出来是几号座，在那撒泼打滚，为了保持现场秩序。放映员范师傅做出决定：张老汉进去随便找个座位坐下，其他的顾客进去后如果你的座位上没人，就坐你自己的座位。如果座位上有人，也随便找个座位坐下。后面的顾客的票根都很清晰。 那么问：第70个进场的2狗子同学他的票是3排2号，他坐在自己座位上的概率是多少？？

  • 出题人:P
  • 答题人:K
  • 出题时间:10:07
  • 答题时间:13:12(用时3分钟)
  • 答题结果:0.5 结果正确
  • 正确答案:50%
  • 解答:因为概率要么就是坐在自己的位置，要么就是坐在别人的位置。数学题有这么一个规律，题目越长越简单。题目越短，反而难。
