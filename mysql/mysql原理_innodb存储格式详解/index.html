1页(page)=基本单位，存储和读取的核心，每页大小默认：16k

1区(extend)=64页,64*16k=1M

1组(space)=256区, 256*1M=256M

每个page都有个编号，整型最大 2^32 x 16k = 64T （单表大小上限）
